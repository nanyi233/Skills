# MBT-pptskills 设计规范

基于现有的 `create-ppt.js` 提炼出的设计系统，确保生成的 PPT 保持一致的视觉风格。

## 配色方案

### 主色调（深色科技风）

```javascript
const C = {
  darkBg: "0B1D3A",       // 深蓝背景（主背景色）
  cardBg: "132E5B",       // 卡片背景（第一层）
  cardBg2: "1A3D72",      // 卡片背景（第二层，稍浅）
  accent: "00B4D8",       // 主色调蓝（强调色、链接、按钮）
  accentLight: "48CAE4",  // 浅蓝（装饰、次要强调）
  gold: "FFB703",         // 金色（高亮、重要信息）
  white: "FFFFFF",        // 白色（主标题、正文）
  lightText: "CAE0F0",    // 浅色文字（副标题、正文）
  muted: "8BA4C7",        // 柔和灰蓝（说明文字、次要信息）
  success: "06D6A0",      // 成功绿（特殊强调）
  gradientLight: "0F2B55" // 渐变用浅色
};
```

### 使用建议

- **背景**：darkBg (#0B1D3A) 作为幻灯片背景
- **卡片**：cardBg (#132E5B) 作为卡片/内容块背景
- **强调**：accent (#00B4D8) 用于按钮、链接、分隔线
- **高亮**：gold (#FFB703) 用于关键数据、重要标题
- **文字层次**：
  - 主标题：white (#FFFFFF)
  - 副标题/正文：lightText (#CAE0F0)
  - 说明/次要：muted (#8BA4C7)

## 布局模式

### 1. 标题页布局

```
装饰圆形（右上角） + 装饰圆形（左下角）
主标题（左上，48pt）
副标题（左侧，18pt）
底部条（tagline）
```

坐标参考：
- 主标题：`x: 0.8, y: 1.5, w: 7, h: 1.2`
- 装饰圆形（右上）：`x: 6.5, y: -1.5, w: 5, h: 5`（半透明 92%）
- 装饰圆形（左下）：`x: -1.5, y: 3.5, w: 4, h: 4`（半透明 92%）

### 2. 三卡片横排

```
[卡片1]  [卡片2]  [卡片3]
```

坐标公式：
```javascript
const cx = 0.6 + i * 3.1;  // i = 0, 1, 2
const cy = 2.3;
const cardW = 2.8;
const cardH = 2.6;
```

卡片结构：
- 圆角矩形背景（`rectRadius: 0.12`）
- 顶部装饰条（`h: 0.06`，accent 或 gold 色）
- **图片区域**（原 emoji 位置）：`y: cy + 0.25, w: 1.0, h: 1.0`
- 卡片标题：`y: cy + 0.85, fontSize: 16`
- 卡片描述：`y: cy + 1.3, fontSize: 12`

### 3. 时间轴布局

```
●───●───●───●───●
│   │   │   │   │
[1] [2] [3] [4] [5]
```

时间轴线：`y: 1.6, w: 8.4, line: { width: 2.5 }`
节点圆形：`w: 0.4, h: 0.4`，填充 accent 色
卡片：`w: 1.7, h: 2.6`，每个间距 `1.85` 英寸

### 5. 左侧引用块 + 右侧列表


## 阴影与装饰

### 卡片阴影

```javascript
function cardShadow() {
  return {
    type: "outer",
    color: "000000",
    blur: 8,
    offset: 3,
    angle: 135,
    opacity: 0.25
  };
}
```

**重要**：每次调用 `cardShadow()` 都生成新对象，不要复用同一个 shadow 对象。

### 装饰元素

**半透明圆形**：
- 用于标题页、结论页的背景装饰
- 尺寸：3-5 英寸直径
- 透明度：92-93%
- 位置：部分溢出幻灯片边界（如 `x: -1.5` 或 `y: -2`）

**分隔线**：
- 粗分隔线：`h: 0.06, fill: accent`
- 细分隔线：`line: { width: 0.8, color: accent }`

**标签条**：
- 左侧竖条：`w: 0.06, h: 0.4, fill: accent`
- 用于标记章节标题

## 图片插入规范

### 占位图使用

所有图片在初始生成时使用统一占位图：
```javascript
slide.addImage({
  path: "./assets/placeholder.png",
  x: cx + 0.9,  // 居中位置
  y: cy + 0.25,
  w: 1.0,       // 1英寸正方形
  h: 1.0,
  sizing: {
    type: "contain",  // 保持比例，填充框内
    w: 1.0,
    h: 1.0
  }
});
```

### 图片尺寸建议

| 位置 | 建议尺寸（英寸） | 用途 |
|------|------------------|------|
| 卡片顶部图标 | 1.0 × 1.0 | 小图标、概念插图 |
| 标题页配图 | 3.0 × 3.0 | 大型装饰图 |
| 内容配图 | 2.0 × 1.5 | 横向配图 |
| 时间轴节点图 | 0.8 × 0.8 | 小图标 |

### 图片编号注释

每个 `addImage` 前添加注释：
```javascript
// IMAGE-01: SLIDE 2, 第1张卡片 - 知识内化图标
slide.addImage({ path: "./assets/placeholder.png", ... });

// IMAGE-02: SLIDE 2, 第2张卡片 - 项目构建图标
slide.addImage({ path: "./assets/placeholder.png", ... });
```

编号格式：`IMAGE-XX`（两位数字）

## 坐标系统

pptxgenjs 使用英寸作为单位，16:9 布局：
- 宽度：10 英寸
- 高度：5.625 英寸

常用起始点：
- 左边距：`x: 0.6` 或 `x: 0.8`
- 顶部边距：`y: 0.4`（章节标签）或 `y: 1.1`（内容起始）

---

## 响应式建议

虽然 PPT 是固定尺寸，但考虑不同投影比例：
- 文字不小于 10pt（确保远距离可读）
- 卡片间距不小于 0.2 英寸（视觉分离）
- 关键内容在安全区域内（避免贴边）

## 典型页面模板

### 封面页
- 背景：darkBg + 装饰圆形
- 主标题：48pt Arial Black，白色
- 副标题：18pt Arial，lightText
- 底部条：tagline 或关键词

### 章节页
- 左侧标签条（accent 色）
- 章节标题：14pt 加粗，accent 色
- 内容区：卡片布局或列表

### 内容页
- 2-5 张卡片（横排或网格）
- 每张卡片：图片 + 标题 + 描述
- 底部可选：总结条

### 结论页
- 大引用（60-72pt）
- 行动建议（居中，金色强调）
- 装饰元素

## 反面教材

❌ **不要做的事情**：
1. 不要使用 emoji（🧠🛠️🎤）—— 用真实图片
2. 不要在深色背景上用浅灰文字（对比度不足）
3. 不要复用 shadow 对象（会导致渲染错误）
4. 不要超过 10pt 以下的文字（除非是脚注）
5. 不要在一页上放超过 5 张卡片（信息过载）

✅ **推荐做法**：
1. 用图片代替 emoji，增强视觉质感
2. 保持文字层次（白色主标题 → lightText 副标题 → muted 说明）
3. 使用 cardShadow() 函数，每次生成新 shadow
4. 关键数据用 gold 色高亮
5. 保持呼吸感（卡片间距、边距充足）
