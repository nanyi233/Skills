## 🎯 毛泽东方法论速查

### 矛盾分析法
**核心原则**: 区分主要矛盾和次要矛盾，分析矛盾转化  
**应用步骤**: 1.识别矛盾 → 2.区分主次 → 3.分析关系 → 4.制定对策  
**关键概念**: 主要矛盾、次要矛盾、矛盾转化、对立统一  
**适用场景**: 问题诊断、决策分析、冲突解决、复杂问题分解  
**命令**: `/mao analyze --method=矛盾 [问题]`

### 实践论方法
**核心原则**: 实践-认识-再实践-再认识循环  
**应用步骤**: 1.实践探索 → 2.总结提升 → 3.指导实践 → 4.循环验证  
**关键概念**: 实践第一、感性认识、理性认识、真理标准  
**适用场景**: 工作方法、学习计划、技能提升、经验总结  
**命令**: `/mao analyze --method=实践 [问题]`

### 调查研究法
**核心原则**: 没有调查就没有发言权  
**应用步骤**: 1.制定计划 → 2.深入实际 → 3.分析研究 → 4.得出结论  
**关键概念**: 第一手材料、周密的调查、正确的调查  
**适用场景**: 市场调研、用户研究、问题诊断、数据收集  
**命令**: `/mao analyze --method=调查 [问题]`

### 战略思维法
**核心原则**: 战略上藐视，战术上重视  
**应用步骤**: 1.全局分析 → 2.阶段划分 → 3.资源集中 → 4.稳步推进  
**关键概念**: 持久战思维、统一战线、根据地建设、运动战  
**适用场景**: 规划制定、竞争分析、长期发展、项目管理  
**命令**: `/mao analyze --method=战略 [问题]`

### 群众路线法
**核心原则**: 从群众中来，到群众中去  
**应用步骤**: 1.深入群众 → 2.集中意见 → 3.形成决策 → 4.指导群众  
**关键概念**: 群众观点、群众利益、群众智慧、为人民服务  
**适用场景**: 团队管理、群众工作、组织建设、民主决策  
**命令**: `/mao analyze --method=群众 [问题]`

### 综合分析法
**核心原则**: 智能选择最适合的方法  
**工作方式**: 基于问题内容智能推荐最佳方法  
**推荐准确率**: 78%+ (基于关键词匹配和语义分析)  
**适用场景**: 复杂问题、多维度分析、新手使用  
**命令**: `/mao [问题]` 或 `/mao analyze [问题]`

## 🔧 高级使用技巧

### 1. 问题提法优化
- **具体化**: "如何具体开展用户调研" > "怎么办"
- **情境化**: 提供具体背景信息
- **聚焦化**: 一次只问一个核心问题
- **结构化**: 将复杂问题分解为多个子问题

### 2. 分析方法组合
```bash
# 分阶段分析复杂问题
/mao analyze --method=矛盾 识别主要问题
/mao analyze --method=实践 制定解决方案
/mao analyze --method=战略 规划实施步骤

# 多角度分析同一问题
/mao analyze --method=矛盾 从矛盾角度分析
/mao analyze --method=群众 从群众角度分析
/mao analyze --method=战略 从战略角度分析
```

### 3. 学习与查询结合
