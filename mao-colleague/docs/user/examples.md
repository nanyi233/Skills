### 示例1: 新产品开发决策 (矛盾分析法)

**问题**: 分析新产品开发面临的主要矛盾

```bash
# 使用矛盾分析法
/mao analyze --method=矛盾 分析新产品开发面临的主要矛盾
```

**预期响应**:
```
## 🔍 矛盾分析: 新产品开发面临的主要矛盾

**主要矛盾**: 产品创新与市场需求之间的匹配问题
**次要矛盾**: 
1. 技术实现与成本控制的矛盾
2. 开发进度与质量保证的矛盾
3. 团队协作与资源分配的矛盾

**分析**: 新产品开发的主要矛盾是创新方向与市场实际需求的矛盾。次要矛盾围绕这一主要矛盾展开，包括技术可行性、成本效益、时间质量等。

**解决建议**:
1. 加强市场调研，确保创新方向符合实际需求
2. 建立快速迭代机制，小步快跑，及时调整
3. 合理分配资源，平衡技术、成本、时间、质量的关系

> *"研究任何过程，如果是存在着两个以上矛盾的复杂过程的话，就要用全力找出它的主要矛盾。"*
```


**预期响应**:
```
## 🔍 矛盾分析: 团队绩效不高的主要原因

**矛盾识别**:
- **主要矛盾**: 团队目标与成员能力的矛盾
- **次要矛盾**: 
  1. 个人目标与团队目标的矛盾
  2. 工作量与资源的矛盾
  3. 沟通协作与工作方式的矛盾
  4. 激励机制与工作动力的矛盾

**矛盾分析**:
团队绩效问题往往是多种矛盾交织的结果。主要矛盾通常是团队期望与实际能力之间的差距，次要矛盾围绕这一主要矛盾展开，包括目标对齐、资源分配、协作机制等。

**解决策略**:
1. **针对主要矛盾**: 加强能力培训或调整目标设定
2. **协调次要矛盾**: 建立清晰的沟通机制、合理的资源分配、有效的激励机制
3. **促进矛盾转化**: 将消极矛盾转化为积极动力

> *"在复杂的事物的发展过程中，有许多的矛盾存在，其中必有一种是主要的矛盾，由于它的存在和发展规定或影响着其他矛盾的存在和发展。"*
```


**预期响应**:
```
## 📚 概念查询: 实践论

**概念定义**:
实践论是毛泽东哲学思想的重要组成部分，强调实践在认识过程中的决定性作用。核心观点包括：实践是认识的来源、实践是认识发展的动力、实践是检验真理的标准、实践是认识的目的。

**毛泽东原著引用**:
> "通过实践而发现真理，又通过实践而证实真理和发展真理。实践、认识、再实践、再认识，这种形式，循环往复以至无穷，而实践和认识之每一循环的内容，都比较地进到了高一级的程度。"

**相关概念**:
- **实践**: 人们改造客观世界的一切活动
```
