# 毛泽东.skill 详细使用指南

## 🎯 概述

毛泽东.skill 是一个基于毛泽东方法论和人格风格的AI智能助手，采用现代化六层认知架构。本指南将详细介绍如何使用毛泽东.skill的各项功能，帮助您充分发挥其潜力。

## 🏗️ 六层认知架构简介

毛泽东.skill采用现代化六层认知架构设计，确保系统的稳定性、扩展性和智能化水平：

### 架构层次
```
┌─────────────────────────────────────────────┐
│            用户界面层 (UI Layer)            │
│  • 命令解析、响应格式化、用户交互          │

                       ↓

└──────────────────────┬──────────────────────┘
                       ↓
┌─────────────────────────────────────────────┐
│          方法执行层 (Method Layer)          │

└──────────────────────┬──────────────────────┘
                       ↓
┌─────────────────────────────────────────────┐

                       ↓
```

- **向后兼容**: 完整支持所有老命令，平滑过渡

## 3. 智能分析系统

### 3.1 支持的分析方法

毛泽东.skill支持6种核心分析方法：

#### 1. 矛盾分析法
- **核心原则**: 区分主要矛盾和次要矛盾
- **适用场景**: 问题诊断、决策分析、冲突解决
- **关键概念**: 主要矛盾、次要矛盾、矛盾转化、对立统一
- **应用步骤**: 识别矛盾 → 区分主次 → 分析关系 → 制定对策

#### 2. 实践论方法
- **核心原则**: 实践-认识-再实践循环
- **适用场景**: 工作方法、学习计划、技能提升
- **关键概念**: 实践第一、感性认识、理性认识、真理标准
- **应用步骤**: 实践探索 → 总结提升 → 指导实践 → 循环验证

#### 3. 调查研究法
- **核心原则**: 没有调查就没有发言权
- **适用场景**: 市场调研、用户研究、问题诊断
- **关键概念**: 第一手材料、周密的调查、正确的调查
- **应用步骤**: 制定计划 → 深入实际 → 分析研究 → 得出结论

#### 4. 战略思维法
- **核心原则**: 战略上藐视，战术上重视
- **适用场景**: 规划制定、竞争分析、长期发展
- **关键概念**: 持久战思维、统一战线、根据地建设
- **应用步骤**: 全局分析 → 阶段划分 → 资源集中 → 稳步推进

#### 5. 群众路线法
- **核心原则**: 从群众中来，到群众中去
- **适用场景**: 团队管理、群众工作、组织建设
- **关键概念**: 群众观点、群众利益、群众智慧
- **应用步骤**: 深入群众 → 集中意见 → 形成决策 → 指导群众

#### 6. 综合分析法
- **核心原则**: 智能选择最适合的方法
- **适用场景**: 复杂问题、多维度分析
- **工作方式**: 基于问题内容智能推荐最佳方法
- **推荐准确率**: 78%+（基于关键词匹配和语义分析）

### 3.2 智能推荐机制

系统基于问题内容自动推荐最佳分析方法：

1. **关键词提取**: 从问题中提取关键术语
2. **方法匹配**: 计算方法与问题的匹配度
3. **置信度评估**: 评估推荐方法的可信度
4. **智能推荐**: 输出推荐方法和理由

**示例**:
```
问题: "分析公司部门协作矛盾"
推荐: "矛盾分析法" (78%置信度)
理由: 问题中包含"矛盾"、"协作"、"问题"等关键词
```

典型的分析响应包含以下部分：

```
## 🔍 问题分析: [问题标题]

---

```

## 6. 高级使用技巧

### 6.1 复杂问题分析策略

#### 分阶段分析
```bash
# 第一阶段：矛盾识别
/mao analyze --method=矛盾 识别问题中的主要矛盾

# 第二阶段：实践指导
/mao analyze --method=实践 制定解决方案

# 第三阶段：战略规划
/mao analyze --method=战略 制定实施计划
```

#### 多维度分析
```bash
# 从不同角度分析同一问题
/mao analyze --method=矛盾 从矛盾角度分析
/mao analyze --method=群众 从群众角度分析
/mao analyze --method=战略 从战略角度分析
```

### 6.2 学习路径优化

#### 个性化学习计划
1. **评估当前水平**: 使用`/mao learn progress`查看当前水平
2. **选择合适路径**: 基于评估结果选择学习路径
3. **制定学习计划**: 确定学习时间和频率
4. **定期复习巩固**: 使用`/mao learn recommendations`获取复习建议

#### 专题深度学习
```bash
# 专题学习：矛盾论
/mao learn 矛盾论
/mao concepts 矛盾
/mao analyze --method=矛盾 [实际问题]
```

### 6.3 概念网络深度利用

#### 概念关联分析
1. **查询核心概念**: `/mao concepts [核心概念]`
2. **探索相关概念**: 查看概念解释中的"相关概念"
3. **建立概念网络**: 理解概念之间的关系
4. **应用概念体系**: 将概念网络应用于实际问题分析

#### 方法论概念整合
将方法论与具体概念结合，提升分析深度：
- 矛盾分析法 + 主要矛盾概念
- 实践论方法 + 感性认识/理性认识概念
- 群众路线法 + 群众观点/群众利益概念

### 6.4 性能优化建议

#### 提高响应速度
1. **启用缓存**: 系统默认启用智能缓存
2. **简化问题**: 避免过于复杂或模糊的问题表述
3. **使用专门命令**: 直接使用`/mao analyze --method=[方法]`避免推荐计算

#### 提升分析质量
1. **提供充分背景**: 在问题中提供足够的背景信息
2. **使用具体问题**: 避免过于抽象或笼统的问题
3. **分步骤提问**: 将复杂问题分解为多个子问题
