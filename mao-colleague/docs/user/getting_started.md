# 毛泽东.skill 快速入门指南

## 🎯 简介

毛泽东.skill 是一个基于毛泽东方法论和人格风格的AI智能助手，采用现代化六层认知架构。它能够：
- 运用毛泽东的矛盾分析法、实践论等6种核心方法论智能分析问题
- 提供四级渐进学习系统，支持从入门到专业的学习路径
- 查询674个毛泽东核心概念库，探索概念关系网络
- 以毛泽东的语言风格和思维方式进行交流

## 🏗️ 六层认知架构

毛泽东.skill采用现代化六层认知架构设计：
1. **用户界面层** - 统一的命令接口和响应格式化
2. **分析决策层** - 智能推荐最佳分析方法（准确率78%+）
3. **方法执行层** - 6种具体分析方法执行
4. **知识检索层** - 概念查询和四级学习路径管理
5. **性能监控层** - 实时性能监控和缓存优化
6. **集成接口层** - 统一接口和向后兼容性保障

**版本**: 1.2.0 (六层架构版)
**发布日期**: 2026-04-09

## 🚀 快速开始

### 1. 安装与配置

#### 完整项目安装（推荐）
```bash
# OpenClaw用户
git clone https://github.com/wwwaapplleecu-source/mao-skill ~/.openclaw/workspace/skills/mao-colleague

# Claude Code用户
git clone https://github.com/wwwaapplleecu-source/mao-skill .claude/skills/mao-colleague

# 安装Python依赖（如需要）
pip install -r requirements.txt
```

> **📦 项目即Skill**: 整个`mao-colleague`项目就是一个完整的毛泽东.skill，安装后即可直接使用所有功能。

### 2. 核心命令架构

毛泽东.skill采用统一的主命令+子命令架构，极大简化使用：

```bash
# 快捷方式（智能分析）
/mao [你的问题]

# 完整命令架构
/mao help                      # 获取帮助（智能引导）
/mao analyze [问题]            # 智能分析（支持--method参数）
/mao learn                     # 学习系统（四级渐进路径）
/mao concepts                  # 概念查询系统（674+概念）
/mao compare                   # 方法比较系统
/mao settings                  # 个性化设置系统
```

### 3. 智能分析示例

#### 智能推荐分析
```bash
/mao 分析公司部门协作矛盾

# 系统会自动推荐最佳分析方法（如矛盾分析法，78%置信度）
```

#### 指定方法分析
```bash
/mao analyze --method=矛盾 识别项目中的主要矛盾
/mao analyze --method=实践 如何改进工作流程
/mao analyze --method=调查 设计用户调研方案
```

#### 开始学习
```bash
/mao learn                     # 开始学习（智能推荐路径）
/mao learn --path=入门        # 选择入门路径（15分钟）
/mao learn --path=基础        # 选择基础路径（1小时）
/mao learn --path=进阶        # 选择进阶路径（3小时）
/mao learn --path=专业        # 选择专业路径（10小时）
```

#### 概念查询
```bash
/mao concepts                  # 查看核心概念列表（674+概念）
/mao concepts 矛盾            # 查看"矛盾"概念详细解释
/mao concepts --search=群众   # 搜索包含"群众"的相关概念
```

## 🎭 理解毛泽东.skill的输出风格

### 1. 方法论特征
- **矛盾分析法**: 区分主要矛盾和次要矛盾，分析矛盾转化
- **实践论思维**: 强调实践-认识-再实践-再认识的循环
- **调查研究法**: 没有调查就没有发言权，典型与普遍结合
- **群众路线法**: 从群众中来，到群众中去
- **战略思维法**: 战略上藐视，战术上重视，持久战思维
- **综合分析法**: 智能选择最适合的分析方法

### 2. 语言风格特征
- **同志们开头**: 常用"同志们，"作为开头
- **排比句式**: 使用"既要...又要...还要..."等排比
- **经典比喻**: 使用"纸老虎"、"解剖麻雀"、"星星之火"等毛泽东经典比喻
- **革命术语**: 使用"革命"、"斗争"、"胜利"等术语
- **总结升华**: 在分析后给出总结和行动建议

### 3. 现代化特性
- **智能推荐**: 基于问题内容自动推荐最佳分析方法
- **学习系统**: 四级渐进学习路径，个性化进度跟踪
- **概念网络**: 674个概念的关系网络查询
- **性能监控**: 内置性能监控和缓存优化
- **向后兼容**: 所有老命令继续支持

## 🔧 高级使用技巧

### 1. 问题提法优化
- **具体化**: 不要问"怎么办"，要问"如何具体开展..."
- **情境化**: 描述具体情境和背景信息
- **聚焦化**: 一次只问一个核心问题
- **结构化**: 将复杂问题分解为多个子问题

### 2. 命令组合使用
