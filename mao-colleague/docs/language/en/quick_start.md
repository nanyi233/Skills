# Mao Zedong.skill - Quick Start Guide (English)

## 🎯 Introduction

Mao Zedong.skill is an AI assistant based on Mao Zedong's methodology and personal style, featuring a modern six-layer cognitive architecture. It provides:

- Intelligent analysis using 6 core Mao Zedong methodologies
- Four-level progressive learning system
- 674 concept library with relationship exploration
- Mao Zedong's language style and thinking patterns

## 🏗️ Six-Layer Cognitive Architecture

Mao Zedong.skill adopts a modern six-layer cognitive architecture:

1. **User Interface Layer** - Command parsing and response formatting
2. **Analytics Decision Layer** - Intelligent recommendation (78%+ accuracy)
3. **Method Execution Layer** - 6 specific analysis methods
4. **Knowledge Retrieval Layer** - 674 concepts and four-level learning paths
5. **Performance Monitoring Layer** - Real-time monitoring and cache optimization
6. **Integration Interface Layer** - Unified interface and backward compatibility

**Version**: 1.2.0 (Six-Layer Architecture Edition)
**Release Date**: 2026-04-09

## 🚀 Quick Start

### 1. Installation & Configuration

#### Complete Project Installation (Recommended)
```bash
# OpenClaw users
git clone https://github.com/wwwaapplleecu-source/mao-skill ~/.openclaw/workspace/skills/mao-colleague

# Claude Code users
git clone https://github.com/wwwaapplleecu-source/mao-skill .claude/skills/mao-colleague

# Install Python dependencies (if needed)
pip install -r requirements.txt
```

> **📦 Project as Skill**: The entire `mao-colleague` project is a complete Mao Zedong.skill. Install it once and use all features immediately.

### 2. Core Command Architecture

Mao Zedong.skill uses a unified main-command + sub-command architecture:

```bash
# Shortcut (intelligent analysis)
/mao [your question]

# Complete command architecture
/mao help                      # Get help (intelligent guidance)
/mao analyze [question]        # Intelligent analysis (supports --method parameter)
/mao learn                     # Learning system (four-level progressive paths)
/mao concepts                  # Concept query system (674+ concepts)
/mao compare                   # Method comparison system
/mao settings                  # Personalized settings system
```

### 3. Intelligent Analysis Examples

#### Intelligent Recommendation
```bash
/mao analyze collaboration conflicts in company departments

# System automatically recommends the best method (e.g., Contradiction Analysis, 78% confidence)
```

#### Specify Method Analysis
```bash
/mao analyze --method=contradiction identify main contradictions in the project
/mao analyze --method=practice how to improve workflow
/mao analyze --method=investigation design user research plan
```

#### Start Learning
```bash

/mao learn --path=basic        # Choose basic path (1 hour)
```



## 🎭 Understanding Mao Zedong.skill Output Style

### 1. Methodological Features
- **Contradiction Analysis**: Distinguish main and secondary contradictions, analyze contradiction transformation
- **Practice Theory Thinking**: Emphasize practice-cognition-repractice-recognition cycles
- **Investigation Research**: No investigation, no right to speak; combine typical and universal cases
- **Mass Line**: From the masses, to the masses
- **Strategic Thinking**: Strategically despise, tactically value; protracted war thinking
- **Comprehensive Analysis**: Intelligently select the most suitable analysis method

### 2. Language Style Features
- **Comrades opening**: Often starts with "Comrades,"
- **Parallel structures**: Use "both...and...and also..." parallel patterns
- **Classic metaphors**: Use Mao Zedong's classic metaphors like "paper tiger," "dissecting sparrows"
- **Revolutionary terminology**: Use terms like "revolution," "struggle," "victory"
- **Summary elevation**: Provide summary and action suggestions after analysis

### 3. Modern Features
- **Intelligent recommendation**: Automatically recommends best analysis methods based on question content
- **Learning system**: Four-level progressive learning paths, personalized progress tracking
- **Concept network**: 674 concept relationship network queries
- **Performance monitoring**: Built-in performance monitoring and cache optimization
- **Backward compatibility**: All old commands continue to be supported
