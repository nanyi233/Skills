# Mao Zedong.skill - Guía de Inicio Rápido (Español)

## 🎯 Introducción

Mao Zedong.skill es un asistente de IA basado en la metodología y el estilo personal de Mao Zedong, que cuenta con una arquitectura cognitiva moderna de seis capas. Proporciona:

- Análisis inteligente utilizando 6 metodologías principales de Mao Zedong
- Sistema de aprendizaje progresivo de cuatro niveles
- Biblioteca de 674 conceptos con exploración de relaciones
- Estilo de lenguaje y patrones de pensamiento de Mao Zedong

## 🏗️ Arquitectura Cognitiva de Seis Capas

Mao Zedong.skill adopta una arquitectura cognitiva moderna de seis capas:

1. **Capa de Interfaz de Usuario** - Análisis de comandos y formato de respuesta
2. **Capa de Decisión Analítica** - Recomendación inteligente (78%+ precisión)
3. **Capa de Ejecución de Métodos** - 6 métodos de análisis específicos
4. **Capa de Recuperación de Conocimiento** - 674 conceptos y rutas de aprendizaje de cuatro niveles
5. **Capa de Monitoreo de Rendimiento** - Monitoreo en tiempo real y optimización de caché
6. **Capa de Interfaz de Integración** - Interfaz unificada y compatibilidad con versiones anteriores

**Versión**: 1.2.0 (Edición de Arquitectura de Seis Capas)
**Fecha de Lanzamiento**: 9 de abril de 2026

## 🚀 Inicio Rápido

### 1. Instalación y Configuración

#### Instalación Completa del Proyecto (Recomendado)
```bash
# Usuarios de OpenClaw
git clone https://github.com/wwwaapplleecu-source/mao-skill ~/.openclaw/workspace/skills/mao-colleague

# Usuarios de Claude Code
git clone https://github.com/wwwaapplleecu-source/mao-skill .claude/skills/mao-colleague

# Instalar dependencias de Python (si es necesario)
pip install -r requirements.txt
```

> **📦 Proyecto como Skill**: Todo el proyecto `mao-colleague` es un Mao Zedong.skill completo. Instálalo una vez y usa todas las funciones inmediatamente.

### 2. Arquitectura de Comandos Principales

Mao Zedong.skill utiliza una arquitectura unificada de comando principal + subcomando:

```bash
# Atajo (análisis inteligente)
/mao [su pregunta]

# Arquitectura completa de comandos
/mao help                      # Obtener ayuda (guía inteligente)
/mao analyze [pregunta]       # Análisis inteligente (admite parámetro --method)
/mao learn                     # Sistema de aprendizaje (rutas progresivas de cuatro niveles)
/mao concepts                  # Sistema de consulta de conceptos (674+ conceptos)
/mao compare                   # Sistema de comparación de métodos
/mao settings                  # Sistema de configuración personalizada
```

### 3. Ejemplos de Análisis Inteligente

#### Recomendación Inteligente
```bash
/mao analizar conflictos de colaboración en departamentos de la empresa

# El sistema recomienda automáticamente el mejor método (ej.: Análisis de Contradicciones, 78% confianza)
```

#### Análisis de Método Específico
```bash
/mao analyze --method=contradiction identificar contradicciones principales en el proyecto
/mao analyze --method=practice cómo mejorar el flujo de trabajo
/mao analyze --method=investigation diseñar plan de investigación de usuarios
```

#### Comenzar a Aprender
```bash
/mao learn                     # Comenzar a aprender (recomendación de ruta inteligente)
/mao learn --path=beginner     # Elegir ruta para principiantes (15 minutos)
/mao learn --path=basic        # Elegir ruta básica (1 hora)
/mao learn --path=advanced     # Elegir ruta avanzada (3 horas)
/mao learn --path=professional # Elegir ruta profesional (10 horas)
```

#### Consulta de Conceptos
```bash
/mao concepts                  # Ver lista de conceptos principales (674+ conceptos)
/mao concepts contradiction    # Ver explicación detallada de "contradicción"
/mao concepts --search=mass    # Buscar conceptos que contengan "masa"
```

## 📖 Ejemplos de Uso

### Ejemplo 1: Análisis Inteligente de Contradicciones
```
Usuario: /mao analizar conflictos de colaboración del equipo

Mao Zedong.skill: Primero identificar la contradicción principal. La contradicción principal en la colaboración del equipo está entre los objetivos individuales y los objetivos colectivos, mientras que las contradicciones secundarias incluyen comunicación deficiente, distribución desigual de recursos y definición de roles poco clara. Para promover la transformación favorable de contradicciones, establecer objetivos comunes, fortalecer la comunicación y coordinación, y aclarar la división de roles.
```

### Ejemplo 2: Guía del Método de la Teoría de la Práctica
```
Usuario: /mao analyze --method=practice cómo mejorar el diseño de productos

Mao Zedong.skill: Mejorar el diseño de productos debe seguir el ciclo práctica-conocimiento. Primero realizar práctica de diseño a pequeña escala para obtener conocimiento perceptual; resumir experiencias prácticas para formar conocimiento racional; usar conocimiento racional para guiar práctica de diseño a mayor escala; a través de ciclos práctica-conocimiento-nueva práctica perfeccionar gradualmente el diseño.
```

### Ejemplo 3: Guía del Sistema de Aprendizaje
```
Usuario: /mao learn --path=beginner

Mao Zedong.skill: ¡Bienvenido al aprendizaje de la Metodología Mao Zedong! Ruta actual: Ruta para Principiantes (15 minutos). Objetivo de aprendizaje: Comprender los conceptos básicos y el valor de aplicación de la metodología Mao Zedong. Progreso del módulo: 0/3 (0%). Primer módulo: Resumen de la Metodología Mao Zedong. Objetivos de aprendizaje: Establecer comprensión general de la metodología Mao Zedong.
```

## 🎭 Comprender el Estilo de Salida de Mao Zedong.skill

### 1. Características Metodológicas
- **Análisis de Contradicciones**: Distinguir contradicciones principales y secundarias, analizar transformación de contradicciones
- **Pensamiento de Teoría de la Práctica**: Enfatizar ciclos práctica-conocimiento-nueva práctica-nuevo conocimiento
- **Investigación de Encuestas**: Sin investigación no hay derecho a hablar, combinar casos típicos y universales
- **Línea de Masas**: De las masas, a las masas
- **Pensamiento Estratégico**: Despreciar estratégicamente, valorar tácticamente, pensar en guerra prolongada
- **Análisis Integral**: Seleccionar inteligentemente el método de análisis más adecuado

### 2. Características de Estilo de Lenguaje
- **Inicio de Camaradas**: A menudo comienza con "Camaradas,"
- **Estructuras Paralelas**: Usar patrones paralelos como "tanto...como...y también..."
- **Metáforas Clásicas**: Usar metáforas clásicas de Mao Zedong como "tigre de papel", "diseccionar gorriones"
- **Terminología Revolucionaria**: Usar términos como "revolución", "lucha", "victoria"
- **Elevación de Resumen**: Proporcionar resumen y sugerencias de acción después del análisis

### 3. Características Modernas
- **Recomendación Inteligente**: Recomendar automáticamente mejores métodos de análisis basados en el contenido de la pregunta
- **Sistema de Aprendizaje**: Rutas de aprendizaje progresivas de cuatro niveles, seguimiento personalizado del progreso
- **Red de Conceptos**: Consultas de red de relaciones de 674 conceptos
- **Monitoreo de Rendimiento**: Monitoreo de rendimiento integrado y optimización de caché
- **Compatibilidad con Versiones Anteriores**: Todos los comandos antiguos siguen siendo compatibles

## 🔧 Consejos de Uso Avanzado

### 1. Optimización de Formulación de Problemas
- **Ser específico**: No preguntar "qué hacer", preguntar "cómo llevar a cabo específicamente..."
- **Proporcionar contexto**: Describir situaciones específicas e información de fondo
- **Enfocar**: Plantear solo una pregunta principal a la vez
- **Estructurar**: Descomponer problemas complejos en múltiples subproblemas

### 2. Combinación de Comandos
```bash
# Proceso integral de análisis de problemas
/mao analizar problemas principales que enfrenta el proyecto actual            # Análisis inteligente
/mao learn contradiction theory                            # Aprender metodología relacionada
/mao concepts main contradiction                           # Consultar conceptos principales
/mao analyze --method=practice desarrollar solución       # Guía de práctica
```

### 3. Planificación de Rutas de Aprendizaje
```bash
# Ruta de aprendizaje desde cero
/mao learn --path=beginner      # Comprensión rápida de 15 minutos
/mao learn --path=basic         # Aprendizaje sistemático de 1 hora
/mao learn --path=advanced      # Aplicación profunda de 3 horas
/mao learn --path=professional  # Investigación profesional de 10 horas
```
