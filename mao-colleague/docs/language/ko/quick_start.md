# 마오쩌둥.skill - 빠른 시작 가이드 (한국어)

## 🎯 소개

마오쩌둥.skill은 마오쩌둥의 방법론과 개인 스타일을 기반으로 한 AI 어시스턴트로, 현대적인 6계층 인지 아키텍처를 특징으로 합니다. 다음 기능을 제공합니다:

- 6가지 핵심 마오쩌둥 방법론을 사용한 지능형 분석
- 4단계 점진적 학습 시스템
- 관계 탐색이 포함된 674개 개념 라이브러리
- 마오쩌둥의 언어 스타일과 사고 패턴

## 🏗️ 6계층 인지 아키텍처

마오쩌둥.skill은 현대적인 6계층 인지 아키텍처를 채택합니다:

1. **사용자 인터페이스 계층** - 명령어 파싱 및 응답 포맷팅
2. **분석 의사 결정 계층** - 지능형 추천 (78%+ 정확도)
3. **방법 실행 계층** - 6가지 구체적 분석 방법
4. **지식 검색 계층** - 674개 개념 및 4단계 학습 경로
5. **성능 모니터링 계층** - 실시간 모니터링 및 캐시 최적화
6. **통합 인터페이스 계층** - 통합 인터페이스 및 하위 호환성

**버전**: 1.2.0 (6계층 아키텍처 에디션)
**릴리스 날짜**: 2026년 4월 9일

## 🚀 빠른 시작

### 1. 설치 및 구성

#### 완전 프로젝트 설치 (권장)
```bash
# OpenClaw 사용자
git clone https://github.com/wwwaapplleecu-source/mao-skill ~/.openclaw/workspace/skills/mao-colleague

# Claude Code 사용자
git clone https://github.com/wwwaapplleecu-source/mao-skill .claude/skills/mao-colleague

# Python 종속성 설치 (필요한 경우)
pip install -r requirements.txt
```

> **📦 프로젝트 즉 스킬**: `mao-colleague` 프로젝트 전체가 완전한 마오쩌둥.skill입니다. 한 번 설치하면 모든 기능을 즉시 사용할 수 있습니다.

### 2. 핵심 명령어 아키텍처

마오쩌둥.skill은 통합 메인 명령어 + 서브 명령어 아키텍처를 사용합니다:

```bash
# 단축키 (지능형 분석)
/mao [질문]

# 완전 명령어 아키텍처
/mao help                      # 도움말 가져오기 (지능형 안내)
/mao analyze [질문]           # 지능형 분석 (--method 매개변수 지원)
/mao learn                     # 학습 시스템 (4단계 점진적 경로)
/mao concepts                  # 개념 쿼리 시스템 (674+ 개념)
/mao compare                   # 방법 비교 시스템
/mao settings                  # 개인화 설정 시스템
```

### 3. 지능형 분석 예시

#### 지능형 추천
```bash
/mao 회사 부서 간 협업 갈등 분석

# 시스템이 최적 방법을 자동 추천 (예: 모순 분석법, 78% 신뢰도)
```

#### 지정 방법 분석
```bash
/mao analyze --method=contradiction 프로젝트의 주요 모순 식별
/mao analyze --method=practice 업무 흐름 개선 방법
/mao analyze --method=investigation 사용자 조사 계획 설계
```

#### 학습 시작
```bash
/mao learn                     # 학습 시작 (지능형 경로 추천)
/mao learn --path=beginner     # 초보자 경로 선택 (15분)
/mao learn --path=basic        # 기초 경로 선택 (1시간)
/mao learn --path=advanced     # 고급 경로 선택 (3시간)
/mao learn --path=professional # 전문가 경로 선택 (10시간)
```

#### 개념 쿼리
```bash
/mao concepts                  # 핵심 개념 목록 보기 (674+ 개념)
/mao concepts contradiction    # "모순" 개념 상세 설명 보기
/mao concepts --search=mass    # "대중" 관련 개념 검색
```

## 🎭 마오쩌둥.skill 출력 스타일 이해

### 1. 방법론적 특징
- **모순 분석법**: 주요 모순과 부차적 모순 구별, 모순 전환 분석
- **실천론 사고**: 실천-인식-재실천-재인식 순환 강조
- **조사 연구법**: 조사 없이는 발언권 없음, 전형과 보편 결합
- **대중 노선법**: 대중으로부터, 대중에게로
- **전략적 사고**: 전략적으로 경시, 전술적으로 중시, 지속전 사고
- **종합 분석법**: 최적 분석 방법을 지능적으로 선택

### 2. 언어 스타일 특징
- **동지들 시작**: 종종 "동지들,"로 시작
- **병렬 구조**: "…도 …도 …도 …도" 등의 병렬 패턴 사용
- **고전적 비유**: "종이 호랑이", "참새 해부" 등의 마오쩌둥 고전 비유 사용
- **혁명 용어**: "혁명", "투쟁", "승리" 등의 용어 사용
- **요약 승화**: 분석 후 요약과 행동 제안 제공

### 3. 현대적 특징
- **지능형 추천**: 질문 내용에 기반하여 최적 분석 방법 자동 추천
- **학습 시스템**: 4단계 점진적 학습 경로, 개별화 진행률 추적
- **개념 네트워크**: 674개 개념 관계 네트워크 쿼리
- **성능 모니터링**: 내장 성능 모니터링 및 캐시 최적화
- **하위 호환성**: 모든 구 명령어 계속 지원

## 🔧 고급 사용 기술

### 1. 문제 제기 최적화
- **구체화**: "어떻게 해야 하나요"가 아닌 "구체적으로 어떻게 진행하나요"라고 묻기
- **맥락 제공**: 구체적 상황과 배경 정보 서술
- **집중**: 한 번에 하나의 핵심 문제만 묻기
- **구조화**: 복잡한 문제를 여러 하위 문제로 분해

### 2. 명령어 조합
```bash
# 포괄적 문제 분석 프로세스
/mao 현재 프로젝트가 직면한 주요 문제 분석            # 지능형 분석
/mao learn contradiction theory                            # 관련 방법론 학습

/mao analyze --method=practice 해결책 개발                # 실천 지도
```
