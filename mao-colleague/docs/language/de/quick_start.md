# Mao Zedong.skill - Schnellstart-Anleitung (Deutsch)

## 🎯 Einführung

Mao Zedong.skill ist ein KI-Assistent, der auf Mao Zedongs Methodik und persönlichem Stil basiert und eine moderne sechsschichtige kognitive Architektur bietet. Es bietet:

- Intelligente Analyse mit 6 Kernmethoden von Mao Zedong
- Vierstufiges progressives Lernsystem
- 674 Konzeptbibliothek mit Beziehungsforschung
- Mao Zedongs Sprachstil und Denkmuster

## 🏗️ Sechsschichtige kognitive Architektur

Mao Zedong.skill verwendet eine moderne sechsschichtige kognitive Architektur:

1. **Benutzeroberflächenschicht** - Befehlsanalyse und Antwortformatierung
2. **Analysen-Entscheidungsschicht** - Intelligente Empfehlung (78%+ Genauigkeit)
3. **Methoden-Ausführungsschicht** - 6 spezifische Analysemethoden
4. **Wissensabrufschicht** - 674 Konzepte und vierstufige Lernpfade
5. **Leistungsüberwachungsschicht** - Echtzeitüberwachung und Cache-Optimierung
6. **Integrationsschnittstellenschicht** - Einheitliche Schnittstelle und Rückwärtskompatibilität

**Version**: 1.2.0 (Sechsschichtige Architektur Edition)
**Veröffentlichungsdatum**: 9. April 2026

## 🚀 Schnellstart

### 1. Installation & Konfiguration

#### Vollständige Projektinstallation (Empfohlen)
```bash
# OpenClaw-Benutzer
git clone https://github.com/wwwaapplleecu-source/mao-skill ~/.openclaw/workspace/skills/mao-colleague

# Claude Code-Benutzer
git clone https://github.com/wwwaapplleecu-source/mao-skill .claude/skills/mao-colleague

# Python-Abhängigkeiten installieren (falls benötigt)
pip install -r requirements.txt
```

> **📦 Projekt als Skill**: Das gesamte `mao-colleague`-Projekt ist ein vollständiger Mao Zedong.skill. Einmal installiert, können sofort alle Funktionen genutzt werden.

### 2. Kernbefehlsarchitektur

Mao Zedong.skill verwendet eine einheitliche Hauptbefehl + Unterbefehl Architektur:

```bash
# Verknüpfung (intelligente Analyse)
/mao [Ihre Frage]

# Vollständige Befehlsarchitektur
/mao help                      # Hilfe erhalten (intelligente Führung)
/mao analyze [Frage]           # Intelligente Analyse (unterstützt --method Parameter)
/mao learn                     # Lernsystem (vierstufige progressive Pfade)
/mao concepts                  # Konzept-Abfragesystem (674+ Konzepte)
/mao compare                   # Methodenvergleichssystem
/mao settings                  # Personalisiertes Einstellungssystem
```

### 3. Intelligente Analysebeispiele

#### Intelligente Empfehlung
```bash
/mao Zusammenarbeitskonflikte in Unternehmensabteilungen analysieren

# System empfiehlt automatisch beste Methode (z.B. Widerspruchsanalyse, 78% Vertrauen)
```

#### Spezifische Methodenanalyse
```bash
/mao analyze --method=contradiction Hauptwidersprüche im Projekt identifizieren
/mao analyze --method=practice Wie Workflow verbessern
/mao analyze --method=investigation Nutzerforschungsplan entwerfen
```

#### Lernen starten
```bash
/mao learn                     # Lernen starten (intelligenter Pfadempfehlung)
/mao learn --path=beginner     # Anfängerpfad wählen (15 Minuten)
/mao learn --path=basic        # Grundlagenpfad wählen (1 Stunde)
/mao learn --path=advanced     # Fortgeschrittenenpfad wählen (3 Stunden)
/mao learn --path=professional # Profipfad wählen (10 Stunden)
```

#### Konzeptabfrage
```bash
/mao concepts                  # Kernkonzeptliste anzeigen (674+ Konzepte)
/mao concepts contradiction    # Detaillierte Erklärung von "Widerspruch" anzeigen
/mao concepts --search=mass    # "Masse" enthaltende Konzepte suchen
```

## 📖 Verwendungsbeispiele

### Beispiel 1: Intelligente Widerspruchsanalyse
```
Benutzer: /mao Teamzusammenarbeitskonflikte analysieren

Mao Zedong.skill: Zuerst den Hauptwiderspruch identifizieren. Der Hauptwiderspruch in der Teamzusammenarbeit liegt zwischen individuellen Zielen und kollektiven Zielen, während Nebenwidersprüche schlechte Kommunikation, ungleichmäßige Ressourcenverteilung und unklare Rollendefinitionen umfassen. Um die Widerspruchstransformation in günstige Richtung zu fördern, müssen gemeinsame Ziele etabliert, Kommunikation und Koordination gestärkt und Rollenaufteilung geklärt werden.
```

### Beispiel 2: Praxistheorie-Methodeleitung
```
Benutzer: /mao analyze --method=practice Wie Produktdesign verbessern

Mao Zedong.skill: Produktdesignverbesserung sollte dem Praxis-Erkenntnis-Zyklus folgen. Zuerst kleine Designpraxis durchführen, um sinnliche Erkenntnis zu erhalten; Praxiserfahrungen zusammenfassen, um rationale Erkenntnis zu bilden; rationale Erkenntnis verwenden, um größere Designpraxis zu leiten; durch Praxis-Erkenntnis-Wiederpraxis-Zyklen Design schrittweise vervollkommnen.
```

### Beispiel 3: Lernsystemführung
```
Benutzer: /mao learn --path=beginner

Mao Zedong.skill: Willkommen zum Mao Zedong Methodik-Lernen! Aktueller Pfad: Anfängerpfad (15 Minuten). Lernziel: Grundkonzepte und Anwendungswert der Mao Zedong-Methodik verstehen. Modulfortschritt: 0/3 (0%). Erstes Modul: Überblick über Mao Zedong-Methodik. Lernziele: Gesamtverständnis der Mao Zedong-Methodik etablieren.
```

## 🎭 Mao Zedong.skill Ausgabestil verstehen

### 1. Methodologische Merkmale
- **Widerspruchsanalyse**: Haupt- und Nebenwidersprüche unterscheiden, Widerspruchstransformation analysieren
- **Praxistheorie-Denken**: Praxis-Erkenntnis-Wiederpraxis-Wiedererkennungszyklen betonen
- **Untersuchungsforschung**: Ohne Untersuchung kein Recht zu sprechen, typische und universelle Fälle kombinieren
- **Massenlinie**: Von den Massen, zu den Massen
- **Strategisches Denken**: Strategisch verachten, taktisch wertschätzen, langwierigen Krieg denken
- **Umfassende Analyse**: Intelligent geeignetste Analysemethode auswählen

### 2. Sprachstilmerkmale
- **Genossen-Anfang**: Oft beginnend mit "Genossen,"
- **Parallele Strukturen**: "Sowohl...als auch...und auch..." parallele Muster verwenden
- **Klassische Metaphern**: Mao Zedongs klassische Metaphern wie "Papiertiger", "Spatzen sezieren" verwenden
- **Revolutionäre Terminologie**: Begriffe wie "Revolution", "Kampf", "Sieg" verwenden
- **Zusammenfassungssteigerung**: Nach Analyse Zusammenfassung und Handlungsvorschläge bieten

### 3. Moderne Merkmale
- **Intelligente Empfehlung**: Automatisch beste Analysemethoden basierend auf Frageninhalt empfehlen
- **Lernsystem**: Vierstufige progressive Lernpfade, personalisierte Fortschrittsverfolgung
- **Konzeptnetzwerk**: 674 Konzeptbeziehungsnetzwerk-Abfragen
- **Leistungsüberwachung**: Integrierte Leistungsüberwachung und Cache-Optimierung
- **Rückwärtskompatibilität**: Alle alten Befehle weiterhin unterstützt

## 🔧 Erweiterte Verwendungstipps

### 1. Problemformulierungsoptimierung
- **Spezifisch sein**: Nicht "Was tun" fragen, sondern "Wie konkret durchführen..."
- **Kontext bereitstellen**: Spezifische Situationen und Hintergrundinformationen beschreiben
- **Fokussieren**: Nur eine Kernfrage gleichzeitig stellen
- **Strukturieren**: Komplexe Probleme in mehrere Teilprobleme zerlegen

### 2. Befehlszusammenstellung
```bash
# Umfassender Problemanalyseprozess
/mao Hauptprobleme analysieren, die aktuelles Projekt konfrontieren            # Intelligente Analyse
/mao learn contradiction theory                            # Verwandte Methodik lernen
/mao concepts main contradiction                           # Kernkonzepte abfragen
/mao analyze --method=practice Lösung entwickeln           # Praxisleitung
```

### 3. Lernpfadplanung
```bash
# Lernpfad von Null an
/mao learn --path=beginner      # 15-minütiges schnelles Verständnis
/mao learn --path=basic         # 1-stündiges systematisches Lernen
/mao learn --path=advanced      # 3-stündige tiefe Anwendung
/mao learn --path=professional  # 10-stündige professionelle Forschung
```
