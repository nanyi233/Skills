## 🎯 はじめに

Mao Zedong.skillは、毛沢東の方法論と個人スタイルに基づくAIアシスタントで、現代的な6層認知アーキテクチャを備えています。以下の機能を提供します：

- 6つのコア毛沢東方法論を使用したインテリジェント分析
- 4段階の漸進的学習システム
- 関係性探索付き674概念ライブラリ
- 毛沢東の言語スタイルと思考パターン

## 🏗️ 6層認知アーキテクチャ

Mao Zedong.skillは現代的な6層認知アーキテクチャを採用しています：

1. **ユーザーインターフェース層** - コマンド解析と応答フォーマット
2. **分析決定層** - インテリジェント推奨（78％以上の精度）
3. **方法実行層** - 6つの具体的分析方法
4. **知識取得層** - 674概念と4段階学習パス
5. **パフォーマンス監視層** - リアルタイム監視とキャッシュ最適化
6. **統合インターフェース層** - 統一インターフェースと下位互換性

**バージョン**: 1.2.0（6層アーキテクチャ版）
**リリース日**: 2026年4月9日

## 🚀 クイックスタート

### 1. インストールと設定

#### 完全プロジェクトインストール（推奨）
```bash
# OpenClawユーザー
git clone https://github.com/wwwaapplleecu-source/mao-skill ~/.openclaw/workspace/skills/mao-colleague

# Claude Codeユーザー
git clone https://github.com/wwwaapplleecu-source/mao-skill .claude/skills/mao-colleague

# Python依存関係のインストール（必要に応じて）
pip install -r requirements.txt
```

> **📦 プロジェクト即スキル**: `mao-colleague`プロジェクト全体が完全なMao Zedong.skillです。一度インストールすればすべての機能をすぐに使用できます。

### 2. コアコマンドアーキテクチャ

Mao Zedong.skillは統一されたメインコマンド＋サブコマンドアーキテクチャを使用：

```bash
# ショートカット（インテリジェント分析）
/mao [質問]

# 完全コマンドアーキテクチャ
/mao help                      # ヘルプの取得（インテリジェントガイド）
/mao analyze [質問]           # インテリジェント分析（--methodパラメータ対応）
/mao learn                     # 学習システム（4段階漸進的パス）
/mao concepts                  # 概念クエリシステム（674+概念）
/mao compare                   # 方法比較システム
/mao settings                  # パーソナライズ設定システム
```
