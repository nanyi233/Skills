## 💡 使用提示

### 核心命令参考
```
/mao [问题]                    # 智能分析快捷方式
/mao analyze --method=矛盾 [问题] # 指定方法分析
/mao learn                     # 学习系统入口
/mao learn --path=入门         # 开始入门学习
/mao learn 矛盾论              # 专题学习
/mao learn progress            # 查看学习进度
/mao concepts [概念]           # 查询概念
/mao help                      # 获取帮助
```

### 常见问题
1. **命令记不住**：使用 `/mao help` 查看帮助
2. **不知道用什么方法**：直接输入问题，让系统智能推荐
3. **想系统学习**：从 `/mao learn --path=入门` 开始
4. **想查概念**：使用 `/mao concepts [概念名]`

### 测试技巧
1. **大胆尝试**：不要怕出错，错误信息也是测试重点
2. **如实记录**：记录真实的感受和困难
3. **独立思考**：按自己的理解操作，不要问引导性问题
4. **时间控制**：每个任务控制在10分钟内
