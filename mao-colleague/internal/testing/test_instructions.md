# 毛泽东.skill 用户测试说明

## 🎯 测试目的

本次测试旨在验证**毛泽东.skill模块三（用户体验优化）** 的新功能，包括：

1. **新命令架构**：主命令+子命令模式，从12个命令减少到7个
2. **智能推荐系统**：基于问题内容推荐最佳分析方法
3. **学习系统**：渐进式四级学习路径和专题学习
4. **向后兼容**：老命令继续支持，平滑过渡

通过测试，我们希望：
- 验证新架构的易用性和易记性
- 评估智能推荐的准确性和实用性
- 收集学习系统的用户体验反馈
- 发现潜在问题和改进机会

## 🧪 测试内容

### 核心测试项目

| 测试类别 | 测试重点 | 预计时间 |
|----------|----------|----------|
| **智能分析** | 快捷命令、指定方法、纯方法论 | 10分钟 |
| **学习系统** | 路径学习、进度跟踪、专题学习 | 15分钟 |
| **命令体验** | 新架构易用性、向后兼容性 | 5分钟 |
| **综合测试** | 完整工作流程测试 | 10分钟 |

### 推荐测试流程

#### 快速测试（15分钟）
1. **智能分析** (5分钟)
   - `/mao 分析公司部门协作问题`
   - `/mao analyze --method=矛盾 分析团队冲突`
   - `/mao analyze --style=work 制定项目计划`

2. **学习系统** (5分钟)
   - `/mao learn`
   - `/mao learn --path=入门`
   - `/mao learn progress`
   - `/mao learn 矛盾论`

3. **兼容性测试** (5分钟)
   - `/mao-analyze 矛盾 公司内部问题`
   - `/mao-work 分析商业案例`

#### 完整测试（30分钟）
按[用户测试用例](user_test_cases.md)中的所有10个测试用例执行。

## 🚀 测试步骤

### 步骤1：环境验证
1. 打开OpenClaw WebChat或您常用的界面
2. 发送命令: `/mao help`
3. 确认看到新命令架构的帮助信息

### 步骤2：智能分析测试
1. **快捷命令测试**
   ```
   /mao 分析公司部门协作矛盾
   ```
   - 观察：是否智能推荐合适方法？
   - 检查：响应结构是否清晰？

2. **指定方法测试**
   ```
   /mao analyze --method=战略 制定市场进入策略
   ```
   - 观察：是否使用战略分析方法？
   - 检查：步骤是否完整？

3. **纯方法论测试**
   ```
   /mao analyze --style=work 解决团队沟通问题
   ```
   - 观察：是否排除毛泽东人格风格？
   - 检查：是否专注于方法论？

### 步骤3：学习系统测试
1. **系统介绍**
   ```
   /mao learn
   ```
   - 检查：是否清晰展示四级路径？

2. **路径详情**
   ```
   /mao learn --path=基础
   ```
   - 检查：路径信息是否完整？

3. **学习进度**
   ```
   /mao learn progress
   ```
   - 检查：进度显示是否直观？

4. **专题学习**
   ```
   /mao learn 实践论
   ```
   - 检查：专题内容是否全面？

### 步骤4：兼容性测试
1. **老命令测试**
   ```
   /mao-analyze 矛盾 分析公司内部问题
   ```
   - 观察：是否提示迁移到新命令？
   - 检查：功能是否正常？

2. **风格命令测试**
   ```
   /mao-persona 用毛式风格写动员讲话
   ```
   - 检查：人格风格是否保持？

## 🐛 常见问题处理

### 1. 命令无法识别
**可能原因**：
- 命令格式错误
- 毛泽东.skill未正确加载

**解决方案**：
1. 检查命令格式：`/mao [子命令] [参数]`
2. 确认技能已安装：联系管理员
3. 尝试帮助命令：`/mao help`

### 2. 响应时间过长
**可能原因**：
- 网络延迟
- 系统处理复杂

**解决方案**：
1. 等待10-15秒
2. 如超过30秒未响应，记录问题
3. 尝试简化问题重新发送

### 3. 内容错误或不完整
**可能原因**：
- 解析错误
- 知识库限制

**解决方案**：
1. 记录具体错误内容
2. 尝试不同表述方式
3. 在反馈中详细描述

### 4. 学习进度不保存
**说明**：
- 当前版本的学习进度为会话内保存
- 重启会话后进度会重置
- 这是设计限制，非错误
