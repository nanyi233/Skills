## 🔍 资料来源与收集策略

### 1. 官方权威来源
- **毛泽东选集** (1-4卷)：中共中央文献研究室编
- **毛泽东文集** (1-8卷)：人民出版社
- **建国以来毛泽东文稿**：中央文献出版社
- **毛泽东年谱**：中央文献出版社

### 2. 公开数字资源
- 中国共产党新闻网 (cpc.people.com.cn)
- 马克思主义文库 (marxists.org/chinese)
- 毛泽东著作数据库
- 各大学图书馆数字资源

### 3. 收集原则
1. **准确性优先**: 确保内容与官方版本一致
2. **完整性**: 收集完整著作，避免断章取义
3. **规范性**: 统一格式，规范标点，清除乱码
4. **可验证性**: 标注来源，便于核对

## 🛠️ 技术处理流程

### 1. 文本获取与整理
```
获取原始文本 → 校对准确性 → 统一格式 → 保存为Markdown
```

### 2. 文件命名规范
```
中文书名.md
示例：关于正确处理人民内部矛盾的问题.md
```

### 3. 内容格式标准
```markdown
# 标题

**作者**: 毛泽东
**写作时间**: YYYY年MM月
**来源**: [书籍/文集名称]

---

## 正文内容

[著作正文，保持原段落结构]

---

**注释**:
1. 如有必要，添加简要注释
2. 保持学术中立

**关键词**: 关键词1, 关键词2, 关键词3
```

### 4. 质量验证步骤
1. **内容核对**: 对比权威版本，确保无遗漏、无错字
2. **格式检查**: 检查Markdown格式是否规范
3. **编码验证**: 确保UTF-8编码，无乱码
4. **分类验证**: 确保著作归入正确分类目录
