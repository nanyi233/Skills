## 🚀 部署指南

### 1. 快速部署
```bash
# 克隆仓库
git clone https://github.com/wwwaapplleecu-source/mao-skill

# 安装依赖
pip install -r requirements.txt

# 使用新架构
cd mao-skill
python -c "from tools.mao_skill_integration_v2 import MaoSkillIntegrationV2; integration = MaoSkillIntegrationV2()"
```

### 2. 配置选项
```python
# 启用所有功能
integration = MaoSkillIntegrationV2(
    enable_cache=True,      # 启用缓存
    enable_monitoring=True  # 启用性能监控
)

# 获取性能报告
report = integration.get_performance_report()

# 获取统计信息
stats = integration.get_stats()
```

### 3. 命令使用
```bash
# 新命令架构
/mao help                    # 获取帮助
/mao analyze [问题]          # 智能分析
/mao learn --path=入门       # 开始学习
/mao concepts 矛盾           # 概念查询

# 老命令（完全兼容）
/mao-work [问题]
/mao-persona [文本]
/mao-analyze [方法] [问题]
```
