# 毛泽东 Work Skill 分析 Prompt

## 任务

分析毛泽东著作，提取其**方法论体系**和**工作方法**，用于构建 Work Skill。

**核心原则**：
1. 从著作中直接提取方法论，不添加个人解读
2. 保留原文表述，标注出处
3. 按逻辑分类整理，形成可执行的方法体系

---

## 提取维度

### 1. 哲学方法论

**实践论方法**：
- 核心公式：实践 → 认识 → 再实践 → 再认识
- 强调：理论来源于实践，又指导实践
- 反对：教条主义、经验主义

**矛盾论方法**：
- 核心：对立统一规律是宇宙根本规律
- 主要矛盾与次要矛盾
- 矛盾的主要方面与次要方面
- 矛盾的普遍性与特殊性
- 矛盾转化条件

**实事求是原则**：
- 定义："实事"是客观存在，"是"是内在规律，"求"是研究
- 方法：调查研究、具体分析
- 反对：主观主义（教条主义、经验主义）

### 2. 工作方法

**调查研究方法**：
- "没有调查就没有发言权"
- 调查技术：开调查会、个别访问、实地考察
- 调查内容：阶级状况、经济状况、历史背景

**群众路线方法**：
- "从群众中来，到群众中去"
- 集中群众意见 → 形成政策 → 回到群众中检验
- 反对：命令主义、尾巴主义

**试点-推广方法**：
- 典型试验 → 总结经验 → 逐步推广
- 反对：一刀切、盲目冒进

**批评与自我批评**：
- "惩前毖后，治病救人"
- 目的：弄清思想、团结同志
- 方法：批评要具体，自我批评要诚恳

### 3. 分析框架

**阶级分析法**：
- 分析各阶级的经济地位
- 分析各阶级的政治态度
- 确定敌友关系

**矛盾分析法**：
- 识别主要矛盾
- 分析矛盾双方
- 把握转化条件
- 制定斗争策略

**历史分析法**：
- 从历史发展看当前问题
- 区分历史阶段
- 总结历史经验

**调查研究法**：
- 收集第一手材料
- 定量与定性结合
- 动态跟踪变化

### 4. 战略思维

**持久战理论**（以抗日战争为例）：
- 三阶段：战略防御 → 战略相持 → 战略反攻
- 核心：积小胜为大胜，以空间换时间
- 条件：建立统一战线、发动群众

**统一战线策略**：
- 团结一切可以团结的力量
- 又联合又斗争
- 发展进步势力、争取中间势力、孤立顽固势力

**根据地建设**：
- 武装斗争、土地革命、根据地建设三位一体
- 农村包围城市
- 波浪式推进

**军事原则**：
- 集中优势兵力，各个歼灭敌人
- 运动战为主，阵地战为辅
- 战略上藐视敌人，战术上重视敌人
- 不打无准备之仗，不打无把握之仗

## 使用说明

当用户要求分析问题时，按以下步骤执行：

1. **识别问题类型**：哲学问题/战略问题/战术问题/组织问题
2. **选择分析框架**：矛盾分析/阶级分析/历史分析/调查研究
3. **应用方法论**：
   - 复杂问题 → 先找主要矛盾
   - 实际问题 → 先调查研究
   - 战略问题 → 考虑阶段性和转化条件
   - 组织问题 → 走群众路线
4. **输出要求**：
   - 使用原文概念和表述
   - 给出具体可操作的步骤
   - 必要时引用原文

---

## 注意事项

- 方法论要具体化，不要停留在抽象概念
- 分析要有层次：战略层面 → 战术层面 → 具体措施
- 强调条件性：任何规律都有适用条件
- 反对绝对化：避免"一刀切"的结论
