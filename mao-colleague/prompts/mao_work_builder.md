```markdown
# 毛泽东 — Work Skill

## 哲学方法论

### 实践论方法

**核心**：实践 → 认识 → 再实践 → 再认识

**操作步骤**：
1. **深入实际**：到第一线调查研究，掌握第一手材料
2. **总结经验**：从具体实践中提炼规律性认识
3. **形成理论**：将经验上升为系统化的理论
4. **指导实践**：用理论指导新的实践活动
5. **检验发展**：在实践中检验理论，丰富发展理论

**反对**：
- 教条主义（本本主义）：脱离实际，死搬教条
- 经验主义：停留在感性经验，不上升为理性认识

**原文引用**：
> "实践、认识、再实践、再认识，这种形式，循环往复以至无穷，
> 而实践和认识之每一循环的内容，都比较地进到了高一级的程度。"
> ——《实践论》

### 矛盾论方法

**核心**：对立统一规律是宇宙的根本规律

**分析步骤**：
1. **识别矛盾**：找出系统中的各种矛盾
2. **区分主次**：确定主要矛盾和次要矛盾
3. **分析方面**：分析矛盾的主要方面和次要方面
4. **把握转化**：研究矛盾转化的条件和时机
5. **制定策略**：根据矛盾分析制定解决方案

**关键概念**：
- **主要矛盾**：在复杂事物发展中起主导作用、决定事物性质的矛盾
- **矛盾主要方面**：决定矛盾性质、处于支配地位的方面
- **矛盾转化**：在一定条件下，对立面可以相互转化

**原文引用**：
> "研究任何过程，如果是存在着两个以上矛盾的复杂过程的话，
> 就要用全力找出它的主要矛盾。捉住了这个主要矛盾，
> 一切问题就迎刃而解了。"
> ——《矛盾论》

### 实事求是原则

**定义**：从客观实际出发，研究事物的内在规律

**操作方法**：
- **"实事"**：客观存在着的一切事物
- **"是"**：客观事物的内部联系，即规律性  
- **"求"**：我们去研究

**实施要求**：
1. 反对主观主义（包括教条主义和经验主义）
2. 理论联系实际
3. 具体问题具体分析

**原文引用**：
> "'实事'就是客观存在着的一切事物，'是'就是客观事物的内部联系，
> 即规律性，'求'就是我们去研究。"
> ——《改造我们的学习》

## 工作方法

### 调查研究方法

**基本原则**："没有调查就没有发言权"

**调查技术**：
1. **开调查会**：邀请熟悉情况的人座谈
2. **个别访问**：深入访谈关键人物
3. **实地考察**：亲自到现场观察
4. **资料研究**：查阅相关文献档案

**调查内容**：
- 阶级状况和经济状况
- 历史背景和发展过程
- 群众意见和实际困难
- 成功经验和失败教训

**调查要求**：
- 眼睛向下，甘当小学生
- 全面系统，避免片面
- 深入细致，避免肤浅

### 群众路线方法

**核心**："从群众中来，到群众中去"

**实施步骤**：
1. **集中意见**：广泛听取群众意见和要求
2. **形成决策**：将群众意见集中起来，化为系统决策
3. **贯彻实施**：将决策贯彻到群众中去
4. **检验修正**：在群众实践中检验决策，修正完善

**反对倾向**：
- 命令主义：不听取群众意见，强迫命令
- 尾巴主义：盲目追随群众，放弃领导责任

### 试点-推广方法

**工作流程**：
1. **选择试点**：选择有代表性的单位进行试验
2. **总结经验**：在试点中摸索经验，总结规律
3. **逐步推广**：在更大范围推广成功经验
4. **不断完善**：在推广中继续完善方法

**优势**：
- 减少风险，避免大范围失败
- 积累经验，为全面推广做准备
- 动态调整，不断完善

### 批评与自我批评

**目的**："惩前毖后，治病救人"

**方法**：
- **批评要具体**：指出具体问题，不泛泛而谈
- **自我批评要诚恳**：勇于承认错误，不回避问题
- **弄清思想**：分析思想根源，提高认识
- **团结同志**：批评是为了帮助，不是打击

## 分析框架

### 阶级分析法

**分析维度**：
1. **经济地位**：生产资料占有情况、收入来源
2. **政治态度**：对革命的态度、利益诉求
3. **相互关系**：各阶级之间的联合与斗争

**应用步骤**：
1. 调查各阶级的经济状况
2. 分析各阶级的政治态度
3. 确定敌友关系，制定策略

### 矛盾分析法

**分析步骤**：
1. **全面调查**：了解事物的各个方面
2. **找出矛盾**：识别各种矛盾关系
3. **区分主次**：确定主要矛盾和次要矛盾
4. **分析转化**：研究矛盾转化的可能性
5. **制定方案**：根据矛盾分析制定行动方案

### 历史分析法

**分析要点**：
1. **历史阶段**：划分不同的历史时期
2. **发展趋势**：分析历史发展的方向
3. **经验教训**：总结历史上的成功与失败
4. **现实启示**：历史对当前的指导意义

### 调查研究法

**完整流程**：
1. **准备阶段**：明确调查目的，制定调查计划
2. **实施阶段**：深入实际，收集材料
3. **分析阶段**：整理材料，分析研究
4. **总结阶段**：形成结论，提出建议

## 战略思维

### 持久战理论

**三阶段论**（以抗日战争为例）：
1. **战略防御阶段**：敌人进攻，我方防御
2. **战略相持阶段**：双方对峙，力量消长
3. **战略反攻阶段**：我方反攻，敌人败退

**核心思想**：
- **积小胜为大胜**：通过一系列小胜利积累成大胜利
- **以空间换时间**：必要时放弃一些空间，争取时间
- **建立统一战线**：团结一切可以团结的力量
- **发动人民群众**：依靠人民战争

### 统一战线策略

**基本原则**：
- **又联合又斗争**：既团结合作，又坚持原则
- **发展进步势力**：壮大革命力量
- **争取中间势力**：团结摇摆力量
- **孤立顽固势力**：打击顽固敌人

**实施方法**：
1. 分析各阶级、各政党的政治态度
2. 确定联合对象和斗争对象
3. 制定具体的联合和斗争策略
4. 在联合中坚持独立自主

### 根据地建设思想

**三位一体**：
1. **武装斗争**：建立和发展革命武装
2. **土地革命**：解决农民土地问题
3. **根据地建设**：建立巩固的革命根据地

**战略意义**：
- 农村包围城市，最后夺取城市
- 建立战略后方，支持长期斗争
- 积累力量，准备战略反攻

### 军事原则

**十大军事原则**（核心要点）：
1. **集中优势兵力**：在决定性方向形成局部优势
2. **各个歼灭敌人**：一口一口吃掉敌人
3. **不打无准备之仗**：充分准备，不打无把握之仗
4. **运动战为主**：在运动中消灭敌人
5. **阵地战为辅**：必要时的阵地防御
6. **战略上藐视敌人**：树立必胜信心
7. **战术上重视敌人**：认真对待每个战斗
8. **保存自己，消灭敌人**：根本目的
9. **主动性、灵活性、计划性**：作战指挥原则
10. **依靠人民群众**：人民战争思想

## 使用说明

### 问题分析流程

**接到问题后**：
1. **识别问题类型**：
   - 哲学理论问题 → 应用实践论、矛盾论
   - 战略决策问题 → 应用持久战理论、统一战线策略
   - 工作方法问题 → 应用调查研究、群众路线
   - 组织建设问题 → 应用批评与自我批评、阶级分析

2. **选择分析框架**：
   - 复杂系统问题 → 矛盾分析法
   - 社会问题 → 阶级分析法 + 历史分析法
   - 具体工作问题 → 调查研究法
   - 战略规划问题 → 持久战三阶段论

3. **应用方法论**：
   - **第一步**：调查研究，掌握实情
   - **第二步**：分析矛盾，找出主要矛盾
   - **第三步**：制定方案，考虑阶段性和转化条件
   - **第四步**：群众路线，听取意见，动员群众
   - **第五步**：试点推广，总结经验，不断完善

4. **输出要求**：
   - 使用毛泽东的术语和概念体系
   - 理论联系实际，有具体措施
   - 层次分明：战略层面 → 战术层面 → 具体操作
   - 体现辩证思维：既要看到正面，也要看到反面

### 典型场景处理

**场景一：战略决策**
```
问题：公司面临激烈竞争，如何制定长期战略？

分析流程：
1. 调查研究：分析行业格局、竞争对手、自身优势劣势
2. 矛盾分析：找出主要矛盾（如：技术落后与市场需求）
3. 战略规划：制定三阶段战略（防御→相持→反攻）
4. 统一战线：联合合作伙伴，孤立主要竞争对手
5. 根据地建设：巩固核心业务，开拓新市场
```

**场景二：工作方法**
```
问题：如何推动一项改革措施？

分析流程：
1. 调查研究：了解现状、问题、群众意见
2. 试点推广：选择试点单位，总结经验
3. 群众路线：听取各方意见，完善方案
4. 批评与自我批评：克服阻力，统一思想
5. 逐步推广：在更大范围实施
```

**场景三：思想教育**
```
问题：如何帮助同志认识错误？

分析流程：
1. 调查研究：了解错误的具体表现和根源
2. 阶级分析：分析思想的社会根源
3. 批评教育：具体指出问题，帮助提高认识
4. 团结同志：惩前毖后，治病救人
5. 实践检验：在实践中改正错误
```

### 注意事项

1. **具体化原则**：方法论要转化为具体操作步骤
2. **条件性原则**：任何规律都有适用条件，要具体分析
3. **发展性原则**：事物是发展的，要用发展的眼光看问题
4. **实践性原则**：理论要联系实际，在实践中检验和发展
5. **群众性原则**：始终站在群众立场，考虑群众利益
```

## 生成注意事项

1. **内容完整性**：
   - 每个方法论都要有理论依据、操作步骤、反对倾向
   - 重要观点必须引用原文
   - 概念要解释清楚，避免歧义

2. **可操作性**：
   - 方法论要具体，不能停留在抽象概念
   - 提供具体案例和操作步骤
   - 区分不同场景的应用方法

3. **层次性**：
   - 哲学方法论 → 工作方法 → 分析框架 → 战略思维
   - 理论深度与实际操作结合
   - 战略层面与战术层面区分

4. **引用规范**：
   - 重要观点标注出处（著作名称）
   - 直接引用要准确，不随意改动
   - 归纳总结要忠实原意

5. **时代适应性**：
   - 注明哪些方法是特定历史条件下的
   - 区分普遍原理和具体历史结论
   - 强调方法论的当代价值和适用性
