## 文件格式要求

### 文本文件编码
- **必须使用 UTF-8 编码**
- 避免使用 GBK、GB2312 等编码
- 文件扩展名：`.txt` 或 `.md`

### 内容格式
- 每篇文章单独一个文件
- 文件名为文章标题
- 文件开头可以包含元数据（可选）
- 正文使用纯文本，避免复杂格式

### 元数据格式（可选）

在文件开头可以添加 YAML 格式的元数据：

```yaml
---
title: "实践论"
author: "毛泽东"
date: "1937-07"
source: "《毛泽东选集》第一卷"
category: "philosophy"
---
```

## 示例文件：实践论.txt

```text
---
title: "实践论"
author: "毛泽东"
date: "1937-07"
source: "《毛泽东选集》第一卷"
category: "philosophy"
---

实践论
论认识和实践的关系——知和行的关系

（一九三七年七月）

马克思主义者认为人类的生产活动是最基本的实践活动，是决定其他一切活动的东西。人的认识，主要地依赖于物质的生产活动，逐渐地了解自然的现象、自然的性质、自然的规律性、人和自然的关系；而且经过生产活动，也在各种不同程度上逐渐地认识了人和人的一定的相互关系。一切这些知识，离开生产活动是不能得到的。在没有阶级的社会中，每个人以社会一员的资格，同其他社会成员协力，结成一定的生产关系，从事生产活动，以解决人类物质生活问题。在各种阶级的社会中，各阶级的社会成员，则又以各种不同的方式，结成一定的生产关系，从事生产活动，以解决人类物质生活问题。这是人的认识发展的基本来源。

......

实践、认识、再实践、再认识，这种形式，循环往复以至无穷，而实践和认识之每一循环的内容，都比较地进到了高一级的程度。这就是辩证唯物论的全部认识论，这就是辩证唯物论的知行统一观。
```

## 数据来源建议

### 公开来源
1. **中国共产党新闻网**：http://cpc.people.com.cn/GB/64184/
2. **马克思主义文库**：https://www.marxists.org/chinese/maozedong/
3. **国家哲学社会科学文献中心**：https://www.ncpssd.org/

### 书籍扫描
1. 使用 OCR 软件扫描纸质书籍
2. 推荐 OCR 工具：ABBYY FineReader、Adobe Acrobat
3. 扫描后需人工校对，确保准确率

### 注意事项
1. **版权问题**：毛泽东著作为公开出版作品，但需注意具体版本的版权
2. **准确性**：尽量使用权威版本，避免错误
3. **完整性**：重要著作应完整收录，不随意删节
4. **注释处理**：原著中的注释应保留，可放在文件末尾

## 预处理步骤

1. **收集材料**：将著作整理为文本文件
2. **编码转换**：确保所有文件为 UTF-8 编码
3. **文件命名**：按上述目录结构组织
4. **质量检查**：检查文本完整性，修正明显错误
5. **放入目录**：将整理好的文件放入对应目录

## 快速开始

如果你已经有毛泽东著作的文本文件，可以：

```bash
# 1. 创建目录结构
mkdir -p knowledge/{selected_works,methodology,military,letters,speeches}

# 2. 将文本文件放入对应目录
# 例如：将《实践论》放入 methodology/ 目录
cp 实践论.txt knowledge/methodology/

# 3. 运行文本处理器
python3 tools/text_processor.py --input-dir knowledge --output-dir processed
```
