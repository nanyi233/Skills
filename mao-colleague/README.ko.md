## 🎯 소개

Mao Zedong.skill은 마오쩌둥의 방법론과 개인 스타일에 기반한 AI 어시스턴트로, 현대적인 6계층 인지 아키텍처를 갖추고 있습니다. 다음 기능을 제공합니다:

- 6가지 핵심 마오쩌둥 방법론을 사용한 지능형 분석
- 4단계 점진적 학습 시스템
- 관계 탐색 기능이 있는 674개 개념 라이브러리
- 마오쩌둥의 언어 스타일과 사고 패턴

## 🏗️ 6계층 인지 아키텍처

Mao Zedong.skill은 현대적인 6계층 인지 아키텍처를 채택하고 있습니다:

1. **사용자 인터페이스 계층** - 명령어 구문 분석 및 응답 포맷팅
2. **분석 결정 계층** - 지능형 추천 (78% 이상 정확도)
3. **방법 실행 계층** - 6가지 구체적 분석 방법
4. **지식 검색 계층** - 674개 개념과 4단계 학습 경로
5. **성능 모니터링 계층** - 실시간 모니터링 및 캐시 최적화
6. **통합 인터페이스 계층** - 통일된 인터페이스와 이전 버전 호환성

**버전**: 1.2.0 (6계층 아키텍처 에디션)
**출시일**: 2026년 4월 9일

## 🚀 빠른 시작

### 1. 설치 및 설정

#### 완전 프로젝트 설치 (권장)
```bash
# OpenClaw 사용자
git clone https://github.com/wwwaapplleecu-source/mao-skill ~/.openclaw/workspace/skills/mao-colleague

# Claude Code 사용자
git clone https://github.com/wwwaapplleecu-source/mao-skill .claude/skills/mao-colleague

# Python 종속성 설치 (필요한 경우)
pip install -r requirements.txt
```

> **📦 프로젝트 즉 스킬**: `mao-colleague` 프로젝트 전체가 완전한 Mao Zedong.skill입니다. 한 번 설치하면 모든 기능을 즉시 사용할 수 있습니다.

### 2. 코어 명령어 아키텍처

Mao Zedong.skill은 통합된 메인 명령어 + 서브 명령어 아키텍처를 사용합니다:

```bash
# 바로가기 (지능형 분석)
/mao [질문]

# 완전 명령어 아키텍처
/mao help                      # 도움말 보기 (지능형 가이드)
/mao analyze [질문]           # 지능형 분석 (--method 매개변수 지원)
/mao learn                     # 학습 시스템 (4단계 점진적 경로)
/mao concepts                  # 개념 쿼리 시스템 (674+ 개념)
/mao compare                   # 방법 비교 시스템
/mao settings                  # 개인화 설정 시스템
```
